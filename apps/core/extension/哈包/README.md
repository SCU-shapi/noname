## 角色列表

### 耄耋 (maodie)
- **势力**: 哈
- **体力**: 4
- **身份**: 主公
- **技能**:
  - **游击**: 当你使用或打出基本牌后，你可以重铸自己一张牌。
  - **肃纪**: 出牌阶段限一次，你可以观看任意一名角色的手牌，重铸其中任意数量的牌。该技能使用时不会触发其他任何技能的效果。
  - **问答** (觉醒技): 当你的血量降到3以下时立即发动，从耄耋开始依次选择一项：
    1. 指定一名手牌数大于自己的角色，视为对其使用一张杀，并弃置其一张牌
    2. 指定一名手牌数不大于自己的角色，获得其一张牌
    
    随后所有角色获得技能**皮豆**。若你为主公，可在自己回合开始阶段直接发动。
  
  - **皮豆** (衍生技): 出牌阶段限一次，指定任意一名玩家博弈，声明该角色所有红色/黑色牌点数总和更大，失败的角色失去一点体力并可以立即使用一次肃纪，因皮豆而陷入濒死的角色回复一点体力。

### 还乡基 (huanxiangji)
- **势力**: 哇
- **体力**: 3
- **性别**: 男
- **技能**:
  - **地租** (锁定技): 其他角色的出牌阶段结束时，若其身上没有标记烧和抢，则需交给你一张牌。
  - **倒算** (锁定技): 你的回合外，对你用过杀或造成伤害的角色获得标记烧，弃置或获得你的牌的角色获得标记抢。你的准备阶段根据每名角色拥有的标记发动相应技能，并移除对应标记：
    - **烧**: 对其发动烧杀
    - **抢**: 对其发动抢掠
  
  **衍生技**:
  - **烧杀**: 视为对目标使用一张火杀，此杀不可响应。
  - **抢掠**: 获得目标一张牌，然后再弃置目标一张牌。

### 威基 (weiji)
- **势力**: 呜
- **体力**: 3
- **性别**: 男
- **技能**:
  - **威压** (锁定技): 
    - 当你成为其他角色杀或延时锦囊的目标时，其需交给你一张杀或锦囊牌，否则此牌对你无效。
    - 当你使用杀或锦囊牌指定其他目标时，其需交给你一张牌，否则此杀不可被响应，锦囊牌不可被无懈可击。
